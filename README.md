## ðŸ“ Materi Inti yang Dipelajari

Berikut adalah poin-poin utama yang dikupas tuntas dalam kursus ini, yang membentuk fondasi kuat dalam pemrograman JavaScript:

### I. Pengenalan dan Eksekusi Kode
* **Apa itu JavaScript (JS)?:** Sejarah singkat dan perannya (bahasa *Fullstack* untuk *Frontend* dan *Backend*).
* **Lingkungan Eksekusi:** Menjalankan kode JS di *Browser* (melalui HTML) dan menggunakan *Console*.
* **Sintaks Dasar:** Aturan penulisan kode dan penggunaan *statement*.
* **Komentar:** Cara membuat komentar satu baris (`//`) dan multi-baris (`/* */`).

### II. Tipe Data dan Variabel
* **Tipe Data Primitif:** Memahami dan mempraktikkan penggunaan:
    * `Number` (untuk bilangan bulat dan pecahan, termasuk notasi ilmiah).
    * `String` (operasi dan manipulasi teks).
    * `Boolean` (nilai `true` dan `false`).
    * `null` dan `undefined`.
* **Variabel:** Cara mendeklarasikan dan menyimpan data:
    * Penggunaan `var` (untuk konteks lama).
    * Penggunaan `let` (variabel yang nilainya bisa diubah).
    * Penggunaan **`const`** (konstanta, variabel yang nilainya tidak bisa diubah/bersifat *immutable*).
* **Operator:**
    * Operator Aritmatika (`+`, `-`, `*`, `/`, `%`).
    * Operator Perbandingan (`==`, `!=`, `===`, `!==`, `>`, `<`).
    * Operator Logika (`&&`, `||`, `!`).
    * Operator Penugasan (`=`, `+=`, `-=`, dll.).

### III. Struktur Kontrol
* **Percabangan (Conditional):** Mengatur alur program berdasarkan kondisi:
    * Penggunaan `if`, `else if`, dan `else`.
    * Ekspresi `switch case`.
    * *Ternary Operator* (operator kondisi sederhana).
* **Perulangan (Looping):** Mengeksekusi blok kode berulang kali:
    * Perulangan `for` (dengan inisialisasi, kondisi, dan *increment*).
    * Perulangan `while` dan `do while`.
    * Penggunaan `break` dan `continue` untuk mengontrol perulangan.

### IV. Fungsi dan Struktur Data Dasar
* **Function (Fungsi):**
    * Cara membuat dan memanggil fungsi.
    * Parameter dan *Argument*.
    * *Return Value* (mengembalikan nilai).
* **Tipe Data Kompleks:**
    * **Array:** Cara menyimpan koleksi data, indeks, dan manipulasi dasar.
    * **Object:** Cara membuat, mengakses, dan memanipulasi properti objek.
